TECNOLOGIAS

  1. HTML5: Linguagem de marcação de hipertexto, responsável pelo conteúdo que aparece no Site. Como:
  2. - Textos, Tabelas, Imagens e Vídeos.
  3. É indicado através de tags, principalmente da < html> início e final que é com a barra </html>.
  4. CSS3: Estilização do conteúdo, o design para o site ou página. Como:
  5. - Cores, Fontes, Tamanho, Sombras, Posicionamento.
  6. JS: JavaScript utilizado para a parte lógica, cálculos. Como:
  7. - Interatividade, Menus interativos, validação, Popus animação.
